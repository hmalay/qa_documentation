    Installation

    Clone Project

    git clone https://smhossain@bitbucket.org/scholastic/intl-litpro.git

    Java 8

    Install Java 8 from http://www.oracle.com/technetwork/pt/java/javase/downloads/jdk8­downloads­2133151.html

    On OSX, the JDK is located at /Library/Java/JavaVirtualMachines/jdk1.8.0.111.jdk/Contents

    IDE

    Install Eclipse, based on your operating system, from http://www.eclipse.org/downloads/packages/eclipse­ide­java­ee­developers/neon2
